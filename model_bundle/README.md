# Model Bundle

Place the manuscript-version ShadowIsland model files in this directory.

Expected layout:

```text
model_bundle/paper_model_v1/
  paper_model_v1_encoder.pth
  paper_model_v1_classifier.pth
  paper_model_v1_decoder.pt
  paper_model_v1_feature_scaler.pkl
  paper_model_v1_feature_model.pkl
  paper_model_v1_calibration.bin
```

Model files are distributed as release assets and are intentionally not tracked
in git.
